# אפליקציית מעקב מטבעות קריפטו

אפליקציה למעקב בזמן אמת אחר מטבעות קריפטוגרפיים שונים באמצעות CoinGecko API, עם גרפים בזמן אמת.

## תכונות

- **דף מטבעות**: רשימת מטבעות קריפטו מובילים לפי שווי שוק
- **דף דוחות**: גרף מחירים נוכחיים וגרף 30 יום של ביטקוין
- **דף אודות**: מידע על המפתח והאפליקציה

## טכנולוגיות

- HTML, CSS3, JavaScript (ES6+)
- jQuery 3.6.0
- Chart.js 3.9.1
- Bootstrap 5.3.0
- CoinGecko API
- CryptoCompare API

## התקנה

```bash
git clone https://github.com/eladjak/Jquery-AJAX---Cryptocurrency-API-Tracking-Application-Project.git
```

פתחו את `index.html` בדפדפן.
